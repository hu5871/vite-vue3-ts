## 拉取仓库

你有两种方式下载：

### git

```bash
git clone git@github.com:hu5871/vite-vue3-ts.git
```



### var-cli

var-cli提供了三大框架的模版vue、angular、react

```
npm install -g var-cli
var create my-project-name
```
